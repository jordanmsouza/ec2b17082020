As ferramentas utilizadas para este sistema se destacam pela eficiência e performance no que se trata em ultimas tecnologias já lançadas, para conceitos de IA, Marchine learning e Deep Learning dentre outras, alguns exemplos das ferramentas ao qual iremos utilizar estão a linguagem de programação Phyton, por sua linguagem interpretada e sua boa performance na orientação a objeto, em conjunto utilizaremos também, softwares como Pandas, que é uma biblioteca de software criada para a linguagem Python para manipulação e análise de dados. Em particular, oferece estruturas e operações para manipular tabelas numéricas e séries temporais, tudo isso pelo SDK do openpilot que é um sistema de direção semiautomático de código aberto desenvolvido pela comma.ai e que promove uma interface gráfica para a comunicação com a rede CAN do veículo. Iremos utilizar também o  Autoware, um software de código aberto para veículos autônomos e que tem como função promover um rico conjunto de módulos autônomos compostos de recursos de detecção, computação e que incluem localização, mapeamento, detecção e rastreamento de objetos, reconhecimento de semáforo, planejamento de missão e movimento, geração de trajetória, detecção e seleção de pista, controle de veículos, fusão de sensores, câmeras, LiDARs, RADARs, aprendizado profundo, sistema baseado em regras, navegação conectada , Registro, realidade virtual e assim por diante. Para os meios de pagamento online iremos utilizar APIs de diversos parceiros, que visam garantir maior abrangência para nossos clientes, tais como a API do Mercado Pago, Cielo, dentre outros parceiros.
